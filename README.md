# ESG & Financial Performance Dashboard (Excel)

Interactive Excel dashboard analyzing how financial performance relates to ESG metrics across industries and regions (2015–2025).

![Dashboard Preview](images/excel-dashboard.png)



## Case Study (visual write-up)

https://krrusha.github.io/ESG-Financial-Performance-Analysis/

> Note: To explore the full interactive Excel dashboard, open `ESG_Financial_Performance_Dashboard.xlsx` from this repo.



## Highlights

- Built an Excel dashboard with slicers for **Industry**, **Region**, and **Year** to explore ESG + financial KPIs.
- Compared **Revenue** and **Growth Rate** across regions to identify performance patterns.
- Evaluated the relationship between **Profit Margin** and **ESG Overall Score**.
- Summarized environmental impact using **Carbon Intensity** and **Water Intensity** across industries.
- Visualized **Growth vs P/S** to compare valuation-style signals against growth.



## Tools Used

- **Data Cleaning & Transformation:** Power Query (Excel)
- **Modeling & Analysis:** Power Pivot (Data Model) + DAX + PivotTables
- **Visualization:** PivotCharts + Charts + Slicers (Excel Dashboard)



## Data Source

Kaggle — ESG and Financial Performance Dataset  
https://www.kaggle.com/datasets/shriyashjagtap/esg-and-financial-performance-dataset

> This dataset simulates the financial and ESG (Environmental, Social and Governance) performance of 1,000 global companies across 9 industries and 7 regions from 2015 to 2025. It contains realistic financial metrics (e.g. revenue, profit margins, market capitalization) alongside comprehensive ESG indicators, including carbon emissions, resource usage and detailed ESG scores.
